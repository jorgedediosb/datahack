PRÁCTICA PAG.59 - DOCKERFILE & GIT

• Crea una imagen con Git mediante el archivo Dockerfile:
FROM alpine:3.19.0
RUN apk update
RUN apk add git

• Crea una imagen con un archivo en su estructura
FROM alpine:3.19.0
RUN apk update
ADD http://www.vlsitechnology.org/pharosc_8.4.tar.gz .

• Construir con la etiqueta app_git:v1, ejecutar, entrar y verificar la existencia del archivo.
$ docker build -t app_git:v1 . => el . indica que la ruta del Dockerfile es la misma desde donde ejecutamos.
$ docker run --rm app_git:v1 ls => --rm hace que el contenedor se elimine automáticamente.
