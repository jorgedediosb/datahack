PRÁCTICA 1

1. Ejecutar la imagen de nginx:
$ docker run --name -d mi_contenedor nginx:latest 

# Se podría también con:
$ docker pull nginx -> Descargar imagen nginx:latest
$ docker create mi_contenedor nginx:latest
$ docker star mi_contenedor -> Iniciar el contenedor

2. Exportar el contenedor de nginx con el nombre de nginx.tar
$ docker export -o mi_contenedor_exportado.tar mi_contenedor

3. Importar la imagen del archivo nginx.tar como mynginx
$ docker import mi_contenedor_exportado.tar mynginx

4. Comprobar la correcta importación de la imagen
$ docker images

5. Eliminar la imagen mynginx
$ docker rmi mynginx

6. Eliminar el archivo nginx.tar
$ rm nginx.tar

7. Detener el contenedor nginx
$ docker stop mi_contenedor

8. Eliminar el contenedor nginx
$ docker rm mi_contenedor

9. Eliminar la imagen nginx
$ docker rmi nginx
