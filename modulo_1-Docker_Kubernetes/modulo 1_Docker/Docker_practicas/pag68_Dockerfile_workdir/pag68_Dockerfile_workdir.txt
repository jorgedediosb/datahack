PRÁCTICA 68: DOCKERFILE - WORKDIR

• Crear un archivo dockerfile con el contenido:
FROM alpine:3.17.1
ENV DIRPATH /myfolder
WORKDIR $DIRPATH

• Construir con la etiqueta workdir:v2:
$ docker build -t workdir:v2 .

• Ejecutar con el comando pwd:
$ docker run workdir:v2 pwd

• Construir, con la etiqueta workdir:v3
adaptando el archivo Dockerfile para que el valor de la variable DIRPATH
también pueda obtener el valor desde un argumento:
> Dockerfile:
FROM alpine:3.17.1
ARG DIR_ARG=/myfolder
ENV DIRPATH $DIR_ARG
WORKDIR $DIRPATH

$ docker build -t workdir:v3 --build-arg DIR_ARG=/ruta/deseada .

• Ejecutar con el comando pwd para visualizar el directorio actual:
$ docker run wordir:v3 pwd
