PRÁCTICA PAG 60 - DOCKERFILE - COPY

• Crear un archivo index.html con el contenido: "Bienvenidos al laboratorio de prueba"

• Crear un archivo dockerfile con el contenido:
FROM nginx:alpine
COPY index.html /usr/share/nginx/html/
ENTRYPOINT ["nginx", "-g", "daemon off;"]

• Construir con la etiqueta app_copy:v1, ejecutar y navegar a la página html.
$ docker build -t app_copy:v1 .
$ docker run --rm -d -p 80:80 app_copy:v1
> Navegamos a http://IP_local:80 y vemos mensaje "Bienvenidos al laboratorio de prueba"
> IP_local: $ ipconfig getifaddr en0
